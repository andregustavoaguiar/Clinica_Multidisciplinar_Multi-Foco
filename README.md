## Introdução 🚀

tempo de leitura: 05min

🌟 Antes mesmo de se tornar tema do projeto de extensão da faculdade, criar uma instituição para que crianças neuroatípicas recebam melhores cuidados já era um sonho meu, iniciado com a descoberta de que meu filho é autista. 🧩 A clínica multidisciplinar Multi-Foco está nascendo desse sonho, atender a crescente demanda por tratamento de crianças neuroatípicas com o objetivo de fornecer suporte integrado e abrangente para famílias neuroatípicas, bem-estar e qualidade de vida de todos os envolvidos. A Multi-Foco é uma iniciativa dedicada a oferecer suporte abrangente para famílias com crianças neuroatípicas, reconhecendo a importância de incluir os pais, que também precisam de apoio terapêutico adicional. 🌈❤️

🚀 Atualmente, a Multi-Foco representa o primeiro passo para a criação de um banco de dados arrojado e integrativo entre possíveis clínicas e equipes multidisciplinares, atendendo à necessidade de suporte para famílias com crianças neuroatípicas e promovendo seu desenvolvimento e qualidade de vida. 

☁️ Ao longo do curso, vou aprimorá-la com soluções em nuvem, implementar algoritmos de machine learning para diagnósticos mais precisos e eficazes, algoritmos para características demográficas e previsão de demanda para antecipar o volume de pacientes em determinados períodos... Tudo isso em prol da qualidade de vida para as famílias. ❤️

## Projeto Didático 🎓

📚 Este projeto faz parte do meu aprendizado no curso de Ciência de Dados da Universidade Estácio de Sá, no 1º período, e foi desenvolvido usando todo o conhecimento adquirido ao longo da disciplina. Usei como base as fases de um projeto de banco de dados:
<ol>
    <li>Levantamento de requisitos</li>
    <li>Modelo conceitual</li>
    <li>Modelo lógico</li>
    <li>Modelo físico</li>
</ol>

O objetivo principal do projeto é identificar uma deficiência no meu ambiente/comunidade e auxiliar na melhoria da qualidade de vida, beneficiando pessoas em diversas profissões, ao inserir soluções de banco de dados. Estas soluções ajudarão a abordar problemas complexos de maneira mais eficiente e eficaz. 💪


## Visão Geral

💡 Este projeto de banco de dados visa atender à crescente demanda por tratamento de crianças neuroatípicas e seus tutores na cidade de Araruama/RJ e região. A minha escolha foi inspirada pelas dificuldades enfrentadas por pais de crianças neuroatípicas, ao lidar com clínicas terapêuticas. A falta de compreensão, apoio inadequado e desafios na obtenção de serviços eficazes são questões comuns enfrentadas por muitos pais nessa situação. 🤝

## 🛠️ Instalação e Uso do PGAdmin

Passo a Passo
1. Instalar o PostgreSQL <br>
Baixe e instale o PostgreSQL para o seu sistema operacional a partir do <a href="https://www.pgadmin.org/download/"> site oficial</a>.

2. Criar um Banco de Dados <br>
Abra o terminal ou use o pgAdmin e crie um novo Banco de Dados. Vamos chamá-lo de clinica_exemplo para este exemplo.

No terminal psql<br>
```
CREATE DATABASE clinica_exemplo;    
\c clinica_exemplo                  
```

No pgAdmin: <br>
Clique com o botão direito em "Databases" e selecione "Create" > "Database..."
Nomeie o Banco de Dados como clinica_exemplo

Agora está pronto para prosseguir com a criação das tabelas. 👍

<br>
